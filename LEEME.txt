NUESTRO ESPACIO ULTRA PWA v3.0.0

SUBIR A GITHUB PAGES
1. Descomprime el ZIP.
2. Sube TODOS los archivos al nivel principal del repositorio.
3. Asegúrate de que index.html, manifest.webmanifest, sw.js y los iconos estén juntos.
4. Espera la publicación de GitHub Pages y abre el enlace en Safari.
5. iPhone: Compartir > Agregar a pantalla de inicio.

DATOS
La aplicación mantiene la clave localStorage "nuestroEspacioPro". Al conservar el mismo dominio de GitHub Pages, tus datos existentes continúan disponibles.
